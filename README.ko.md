# Dewdrop Canvas 💧 (물방울 캔버스)

> **물리 기반 액체 인터랙션 마인드맵 & 생성형 AI 생각 융합 보드 (Node.js & Express 기반)**

🌎 **[English Version README](./README.md)**

![Dewdrop Canvas Demo](demo/dewdrop-canvas-demo.gif)

---

**Dewdrop Canvas(물방울 캔버스)**는 아이디어를 스스로 빛나는 유기적인 액체 물방울 형태로 표현하여 시각화해 주는 프리미엄 디지털 마인드맵 보드입니다. 하드웨어 가속 CSS 필터 혼합 기술(`blur(12px) contrast(20)`)과 정밀한 물리 스프링-댐핑 공식(Spring-Mass Damping Physics)을 적용하여, 물방울들을 서로 가까이 드래그하면 실제 물방울처럼 경계면이 쫀득하게 늘어나며 결합하는 환상적인 액체 인터랙션을 선사합니다.

본 프로젝트는 단순 정적 클라이언트 페이지에서 **Node.js Express 서버 기반의 영구 다중 프로젝트 관리자**로 전면 업그레이드되었습니다. 한국어와 영어의 완전한 실시간 다국어 지원, 생각 단어 맞춤형 실시간 색상 분류 시스템, 우클릭 AI 생각 보강 기능, 그리고 캔버스 위에 분산된 생각들을 스스로 분석하고 결합 대상을 지정해 자성으로 끌어당겨 합치는 **AI 자동 시너지 융합 기능**을 탑재하고 있습니다.

## 🌟 프리미엄 핵심 기능

### 1. 📂 지속성 백엔드 다중 프로젝트 엔진 (Multi-Project)
* **독립 프로젝트 관리**: 마인드맵 공간 데이터가 서버 내 `data/projects/` 폴더에 각각 독립된 JSON 파일 데이터베이스로 저장됩니다. 프로젝트의 생성, 이름 변경, 안전한 삭제 관리가 가능합니다.
* **실시간 즉각 자동 저장 (Autosave)**: 물방울의 생성, 드래그 이동, 결합, 삭제 등 캔버스 위에서 일어나는 모든 상호작용이 실시간 백엔드로 즉시 자동 저장됩니다.
* **마지막 편집 상태 자동 복구**: 서버가 사용자의 작업 내역을 추적하여, 브라우저를 새로고침하거나 다시 실행하더라도 마지막으로 열어두었던 프로젝트 캔버스를 그대로 복원합니다.
* **기본 생각물방울 생성 (Seed)**: 새 프로젝트를 생성하면 다국어 설정에 맞추어(한국어: '기본 생각 지도', 영어: 'Default Mind Map') 즉시 만지고 놀 수 있는 아름다운 3개의 기초 아이디어 물방울을 자동으로 생성 및 공급합니다.

### 2. 🎨 내용 맞춤형 실시간 색상 감성 시스템 (Content-Aware Dynamic Color)
* **문맥 의미 및 형태소 파싱**: 사용자가 입력한 생각 단어를 실시간(영어/한국어)으로 분석하여 고유의 HSL 테마 색상을 동적으로 매핑합니다:
  1. **IT / 기술 / 코딩** $\rightarrow$ **시안/네온 블루 테마** (`#06b6d4` / `#6366f1`)
  2. **모닥불 / 장작 / 햇살 / 불꽃** $\rightarrow$ **아늑한 주황/오렌지 테마** (`#f59e0b` / `#f97316`)
  3. **자연 / 숲 / 바람 / 휴식** $\rightarrow$ **에메랄드 초록 테마** (`#10b981` / `#047857`)
  4. **비 / 빗소리 / 바다 / 폭풍** $\rightarrow$ **사파이어 로열 블루 테마** (`#3b82f6` / `#1e3a8a`)
  5. **예술 / 음악 / 꿈 / 시너지** $\rightarrow$ **우주적인 마젠타/보라 테마** (`#a855f7` / `#ec4899`)
  6. **생각 / 메모 / 아이디어 (기본)** $\rightarrow$ **따스한 오렌지/앰버 테마** (`#f97316` / `#d97706` - 경고/에러 연상을 차단하기 위해 붉은색 계열을 따뜻하고 감성적인 골드 앰버 톤으로 매칭)
* **전방위 시각적 동기화**: 매핑된 고유 색상 코드는 캔버스 메타볼, 카드 마우스 오버 테두리, AI 연산 대기용 홀로그램 오라 애니메이션, 그리고 물방울을 터뜨릴 때 터져 나오는 액체 파티클 splash 이펙트까지 완벽히 일치하여 융합됩니다.

### 3. 🪐 AI 자동 시너지 융합 기능 (AI Auto-Synergy Merge Picker)
* **지능형 시너지 추천**: 우측 상단의 우주 별 모양 버튼(`#ai-auto-merge-btn`)을 누르면 백엔드 `POST /api/recommend-merge` API가 호출됩니다. 인공지능이 보드 위에 활성화된 모든 물방울을 분석하고, 결합 시 가장 아름답고 혁신적인 정서적/논리적 시너지를 발휘할 수 있는 두 개의 물방울을 선정합니다.
* **물리적 자성 흡착 스냅**: 두 물방울이 선정되면 즉시 `aiAutoMerging` 물리 상태가 적용되어, 중심점을 향해 스프링 물리 법칙으로 미끄러지듯 마찰력을 감쇄하며(`0.88`) 서로를 강하게 끌어당깁니다.
* **자동 융합 트리거**: 자성으로 미끄러진 두 물방울이 충돌하는 순간, 사용자 개입 없이도 물리 충돌을 즉시 감지하여 생성형 생각 융합(Synthesis Merge) 단계로 자동으로 돌입합니다.

### 4. 🪄 안전한 멀티 LLM 백엔드 융합 프록시 & 단일 생각 보강
* **보안적 API 통신**: 사용자가 입력한 Gemini, DeepSeek, OpenRouter API 키를 안전하게 백엔드 세션 내에서만 중계하여 브라우저 메모리에 노출되지 않도록 전면 보호합니다.
* **톤앤매너(T&M) 조율 프롬프트 엔지니어링**: 실무/기술적인 생각들은 이과형의 사실적이고 명료한 설계 제안서 톤으로 합치고, 예술/웰니스/일상 에세이 톤의 생각들은 유려하고 감성적인 한 편의 문학적 산문체로 정성스럽게 융합합니다.
* **우클릭 "AI 생각 보강 & 정돈"**: 물리 캔버스 위 물방울 카드를 우클릭하면 나타나는 문맥 메뉴를 통해 맞춤법 정리, 표현 정돈 및 AI의 창의적 실천 방안을 1초 만에 덧붙여 물방울 크기를 동적으로 확장할 수 있습니다.

### 5. 🔄 강력한 되돌리기 및 융합 제어 시스템 (`Ctrl + Z`)
* **10단계 히스토리 스택**: 프로젝트별로 최근 10단계의 캔버스 맵 세이브포인트를 유지합니다. 헤더의 되돌리기 버튼 또는 단축키 `Ctrl + Z`를 통해 언제든 안전하게 복구할 수 있습니다.
* **물리 반동 분할 분리 (Immunity)**: 결합된 물방울을 되돌리기(Rollback) 할 시, 두 원래 물방울이 겹쳐서 로딩되는 즉시 자동 재결합되는 악순환을 막기 위해 **2.2초 동안 결합 면역(Merge Immunity) 시간**을 적용하고, 중심점에서 바깥쪽으로 밀어내는 고사양 척력 스프링 반동 속도를 가해 서로 기분 좋게 튕겨 나가 분리되도록 물리 공식을 탑재했습니다.

### 6. 🌐 완전한 언어 설정 및 실시간 UI 현지화 (KO / EN)
* **글로벌 다국어 토글**: 브랜드 설정 모달을 통해 한국어(KO)와 영어(EN)를 자유롭게 전환할 수 있습니다.
* **실시간 UX 로컬라이제이션**: 언어를 바꾸는 즉시 사이드바 정보, 상태 메시지, 다국어 힌트 안내, AI 분석 대기 문장들이 단 1프레임 만에 자연스럽게 동적 번역 적용됩니다.

### 7. 🔊 알고리즘 사운드 & 광활한 보드 공간
* **오프라인 실시간 오디오 신시사이징 (Web Audio API)**: 무거운 사운드 파일 로딩 없이, 순수 웹 오디오 진동 자원을 결합하여 물방울 튀는 소리("Plop!")와 융합 성공 축하 차임벨("Chime") 음향을 실시간 수학적으로 생성 연출합니다.
* **2400px x 1800px 광활한 공간**: 브라우저 창 크기가 줄어들더라도 물방울들이 좁은 공간에 찌부러져 자동 융합되는 것을 완전히 차단해 주는 화면 스크롤 뷰포트 센터링 보호막을 장착하고 있습니다.

## 🚀 로컬 구동 방법

Dewdrop Canvas는 가벼운 백엔드 서버를 통해 매우 신속하고 안전하게 100% 로컬 구동됩니다.

### 1. 패키지 설치
터미널을 열고 프로젝트 폴더 경로로 이동하여 백엔드 구동에 필요한 패키지들을 설치해 주세요:
```bash
# 프로젝트 폴더로 이동
cd C:\Users\kernullist\.gemini\antigravity\scratch\dewdrop-canvas

# 필요한 의존성 라이브러리(Express, CORS, dotenv, node-fetch) 설치
npm install
```

### 2. 인공지능 API 키 설정 (선택사항)
프로젝트 루트 경로 내에 위치한 `.env` 파일을 생성하거나 열어서 원하시는 AI 모델 키를 기입해 줍니다:
```env
PORT=3001
GEMINI_API_KEY=여기에_구글_제미나이_키_입력
DEEPSEEK_API_KEY=여기에_딥시크_키_입력
OPENROUTER_API_KEY=여기에_오픈라우터_키_입력
```
*(API 키를 설정하지 않더라도 최적의 한글 파서 시나리오가 결합된 서버 기반 **Poetic Simulator Mode(감성 시뮬레이터)**가 자동 적용되므로, 모든 물리 충돌 스냅 및 가상의 융합 연산을 즉시 풍부하게 즐기실 수 있습니다.)*

### 3. 백엔드 로컬 데이터 서비스 실행
아래 명령어를 사용하여 로컬 DB 서버를 구동합니다:
```bash
npm start
```

### 4. 브라우저로 캔버스 접속
즐겨 쓰시는 최신 브라우저를 여신 후 다음 주소로 접속해 주세요:
🔗 **[http://localhost:3001](http://localhost:3001)**
